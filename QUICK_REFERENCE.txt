================================================================================
TROUBLESHOOTING
================================================================================

Issue: Can't connect to database
Fix: Check /backend/.env - DB_HOST, DB_USER, DB_PASSWORD

Issue: Port already in use
Fix: Change PORT in /backend/.env or kill process using port

Issue: Module not found
Fix: Delete node_modules and run npm install again

Issue: MySQL not running
Fix: Windows: Services app > Start MySQL Service
     Mac: mysql.server start
     Linux: sudo systemctl start mysql

Issue: Orders not creating
Fix: Run npm run db:init, check backend logs, verify auth token
