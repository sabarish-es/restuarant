================================================================================
                        COMMON ERROR MESSAGES
================================================================================

ERROR: "Username or email already exists"
FIX:   Use unique username and email
       Check if employee already added

ERROR: "Missing required fields: username, email, password, first_name, last_name"
FIX:   Fill in all required fields
       Don't leave any blank

ERROR: "Failed to create employee"
FIX:   Check MySQL is running
       Run: npm run dev

ERROR: "Failed to fetch employees"
FIX:   Backend not responding
       Check http://localhost:3001/api/health

================================================================================
                         DEBUGGING TIPS
================================================================================

1. CHECK BACKEND LOGS
   Look for [v0] messages in terminal:
   [v0] Creating employee with data: {...}
   [v0] User created with ID: 5
   [v0] Employee created with ID: 7

2. TEST DATABASE
   curl http://localhost:3001/api/health
   Should return: {"status":"ok"}

3. REINITIALIZE DATABASE
   npm run db:init

4. BROWSER CONSOLE
   Press F12 → Console tab
   Look for [v0] prefixed messages
